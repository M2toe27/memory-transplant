## 📦 What this is

memory-transplant is a Windows app for moving memory data between Claude accounts. It helps you keep your notes, context, and past memory in one place when you switch accounts.

Use it when you want to:
- move memory from one Claude account to another
- keep useful context after an account change
- save time on manual copy and paste
- keep your memory data under your control

## 🖥️ What you need

Before you start, check these items:
- Windows 10 or Windows 11
- A working internet connection
- Access to both Claude accounts
- Enough free disk space for the app and your exported files
- Permission to run apps from the folder you choose

For best results, use the latest version of Claude in your browser before you begin the transfer.

## 🧭 How to use it

1. Sign in to the Claude account that has the memory you want to move.
2. Open memory-transplant.
3. Choose the source account details inside the app.
4. Connect or sign in to the target Claude account.
5. Review the memory items the app shows.
6. Select the memory you want to move.
7. Start the transfer.
8. Wait for the process to finish.
9. Open the target account and check that the memory appears there.

If your memory list is long, take a moment to review it before you start the transfer. This helps you move the right items.
